Breeze is a WordPress Caching Plugin developed by Cloudways. Breeze uses advance caching systems to improve WordPress loading times exponentially.
 
== Description ==
 
Breeze is a simple and user-friendly WordPress Caching Plugin developed by the Cloudways team. It offers users various options to optimize WordPress performance on various levels.
 
Breeze features excellence on the following levels:.

* **Performance:** Your website speeds will be improved and resources will be optimized. Other features include file level cache system, database cleanup, minification, support for Varnish cache and simplified CDN integration options.

* **Convenience:** It.s easy to install directly from WordPress and configure. Breeze comes pre-configured so you can use it without worrying about configuration. Our recommended settings should work on all your WordPress websites seamlessly. 

* **Simplicity:** It.s quite simple to use for regular and advanced users. Just install and activate the plugin and you.ll see the results instantaneously. 
 
Once you install the plugin, the results will show instantly. Along with awesome site performance, Varnish cache integration will  speed up loading times of your WordPress website exponentially.

**Note:** Not all options will be functional on all hosting providers. For example, Varnish is only functional on hosting providers that have Varnish installed on their servers.

== Installation ==
 
= To install the plugin via WordPress Dashboard: =
* Through WordPress admin, use the menu: Plugin > Add new
* Search for Breeze
* Click on install then click activate link
 
= To install the plugin manually: =
* Download and unzip the plugin cloudways_breeze.zip
* Upload the /cloudways_breeze to /wp-content/plugins/
* Activate the plugin through the 'Plugins' menu in WordPress
* Access Breeze from WordPress Admin > Settings > Breeze

= Is Varnish Supported? =

Breeze supports Varnish. It has been tested  compatible with Cloudways Servers that come with Varnish pre-installed. If you are using any hosting provider other than Cloudways, we suggest that you confirm Varnish support with  them.

= Is WooCommerce Supported? =

Breeze is fully compatible with WooCommerce out of the box. Hence, no special configuration is required.


= Is it Compatible with other WordPress cache plugins? =

It is not recommended to use two WordPress cache plugins on the same time on any WordPress website. We recommend to keep only one Cache plugin active at a time.


= Is it compatible with HTTPS? =

Breeze does not require any special configuration to work with HTTP or HTTPs pages. 

= Does this plugin have incompatibility issues with other known plugins? =

Breeze has been tested with popular plugins available on WordPress.org. However, if you face issues, feel free to report it via WordPress support forums.

= What type of content is supported with CDN? =

Breeze supports CDN integration and allows all the static assets to be served via the CDN, such as images, css and js files. 

= What does Database Optimization do? =

WordPress databases are notorious for storing information like post revisions, spam comments and much more. Over time, databases will become bloated and it is a good practice to clear out unwanted information from the WordPress database to reduce its size and hence optimize it. 

Breeze.s database optimization will help you clear out databases from unwanted information with just a click.

= Will comments and other dynamic parts of my blog appear immediately? =

Breeze is compatible with popular WordPress commenting plugins and also WordPress. default comment system. Whenever a comment is published on a WordPress post page, Cache will automatically be purged for that particular page while keeping cache intact on all other pages.

= Can I exclude URLs of individual files and pages from Cache? =

Yes, by using the exclude fields, you can exclude a file by mentioning its URL, or file type by mentioning file extension. Exclude will not let the cache impact that URL or file type. 

= Does it work with all hosting providers? =

Breeze supports all options including Gzip, browser cache, minification, grouping, database optimization and CDN integration will work similarly on all hosting providers. Varnish, however, is fully compatible with Cloudways servers.
