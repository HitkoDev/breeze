Breeze is a WordPress Caching Plugin that uses advance caching systems to improve WordPress loading time.

== Description ==

Breeze is an easy to use WordPress Caching Plugin developed by a team at Cloudways. It has various options to optimize WordPress performance on different levels.

Breeze has file level cache system, database cleanup, minification, support for Varnish cache and easy CDN integration options for static assets.

Just install the plugin and it will start delivering awesome performance. With use of Varnish cache you can bring down the loading times of your WordPress website by a significant amount.

Note: Not all the options will be functional on all hosting providers. For example Varnish is only functional on hosting providers that have Varnish installed on the servers.

== Installation ==

= To install the plugin via WordPress Dashboard: =
* Through WordPress admin, use the menu: Plugin > Add new
* Search for Breeze
* Click on install then click activate link

= To install the plugin manually: =
* Download and unzip the plugin breeze.zip
* Upload the /breeze to /wp-content/plugins/
* Activate the plugin through the 'Plugins' menu in WordPress
* Access Breeze from WordPress Admin > Settings > Breeze



== Frequently Asked Questions ==

= Is Breeze Compatible with other WordPress cache plugins? =
It is not recommended to use two WordPress cache plugins on the same time on any WordPress website. We recommend to keep only one Cache plugin active at a time.

= Does Breeze Support Varnish Cache? =
Yes, if your hosting provider has Varnish Cache installed on your server, then Breeze will take care of the purging rules of Varnish on your WordPress website.
